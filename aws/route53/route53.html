Records Types

  • A Record: type of Domain Name System (DNS) record that maps a domain name to an IP address.
    • For example, if you have a website hosted at the domain "example.com", you might use an A record to associate the domain name with the IP address of the web server where the website is hosted
  • CNAME Record: type of Domain Name System (DNS) record that maps a domain name to another domain name
    • Used to associate an alias or "canonical" domain name with a primary or "target" domain name.
    • For example, if you have a website hosted at the domain "example.com", you might use a CNAME record to associate the domain name "www.example.com" with the primary domain name "example.com".
  • PTR Record (aka Reverse DNS Record): maps an IP address to a domain name
    • PTR records are typically used for reverse DNS lookups, which are used to determine the domain name associated with a particular IP address
  • Alias Record: is a type of Domain Name System (DNS) record that maps a domain name to another resource, such as an Amazon S3 bucket, an Amazon CloudFront distribution, or an Amazon Elastic Load Balancer
