Notification Rules

  • Notification rules allow you to receive notifications about events that occur in your CodeCommit repository
  • You can set up notification rules to trigger notifications via email, Amazon SNS, or AWS Lambda functions, based on events such as
    • Code commits
    • Pull requests
    • Branch and tag operations
    • Approval rules
    • Comments
