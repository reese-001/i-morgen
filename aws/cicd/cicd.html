Continuous Integration (CI)

  • Developers push the code into a code repo (GitHub, CodeCommit, BitBucket, etc)
  • A test of build server will perform predefined checks as soon as the code is pushed (CodeBuild, Jenkins CI, etc)
  • Developer gets feedback about which tests failed and passed
  • Benefits
    • Find bugs early
    • Deliver code faster
    • Deploy often

Continuous Delivery/Deployment (CD)

  • Ensures that code can be released reliably whenever needed
  • Ensures deployments happen often and quick
  • Automated deployment tools include Jenkins CD, CodeDeploy, Spinnaker, etc
