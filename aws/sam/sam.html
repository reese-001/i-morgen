Serverless Application Model (SAM)

What does it do? 

  • Extension of CloudFormation that lets you define serverless applications
  • SAM is both an AWS CLI took and a CloudFormation Macro. Effortless to define and deploy serverless apps
  • When compared to CloudFormation, SAM allows at least a 50% reduction in code
  • CLI makes it easy to run, package and deploy serverless applications or Lambdas
