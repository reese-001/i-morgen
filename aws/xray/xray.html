X-Ray

What is it?

  • AWS X-Ray provides a complete view of requests as they travel through your application and filters visual data across payloads, functions, traces, services, APIs, and more with no-code and low-code motions.

Xray Daemon

  • The AWS X-Ray daemon is a software application that listens for traffic on UDP port 2000, collects raw segment data, and sends it to the X-Ray API.
  • The daemon works in conjunction with the AWS X-Ray SDKs and must be installed on each host that you want to trace.

Xray Compared to CloudWatch

    • CloudWatch is a monitoring service that provides data and operational insights for various AWS resources
      X-Ray is a debugging and troubleshooting service that helps you understand the performance and behavior of distributed applications
