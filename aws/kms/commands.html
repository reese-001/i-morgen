Common Commands

reference: kms — AWS CLI 1.27.23 Command Reference (amazon.com)


aws kms create-key creates a unique customer managed CMK 
aws kms encrypt encrypts plaintext into ciphertext by using CMK
aws kms decrypt decrypts ciphertext that was encrypted by CMK
aws kms re-encrypt decrypts ciphertext and then re-encrypts it within KMS
  • manually rotate a CMK
  • change the CMK that protects ciphertext
  • change the encryption context of a ciphertext
aws kms enable-key-rotation enable auto rotation of the key material for specified CMK
