Secrets Manager

What does it do? 

  • Protect secrets needed to access your applications and services
  • Easily rotate, manage and retrieve database credentials, API keys, and other secrets through their lifecycle
  • Primary use is to store and rotate database credentials
    • RDS
    • Redshift
    • DocumentDB
    • Other Databases
    • Key Value

Automatic Rotation

  • Can setup automatic rotation for any database
  • Can rotate up to 365 days
  • Performed via Lambda function
  • Can rorate password for superuser or developer programmatically accessing the database

Note: Enforces at rest by using KMS

Note: CloudTrail can monitor credentials access in case of audit

Pricing: $.40 per secret per month / $0.05 per 10k api calls
