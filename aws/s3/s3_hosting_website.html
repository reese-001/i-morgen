S3 Website Hosting

Overview

  • S3 can host static websites and have them available on the Internet
  • Website URL will be (depending on region)
    • http://bucket-name.s3-website-aws-region.amazonaws.com
    • http://bucket-name.s3-website.aws-region.amazonaws.com
  • If you get an http 403, make sure the bucket policy allows public reads

S3 Bucket Setup

  • Navigate into an S3 bucket and select "Properties"
  • Scroll down to bottom and select "Static website hosting
  • Modify the following settings
    • Select "Enable" for Static website hosting
    • Hosting type: Host a static website
    • Index document: As required, or index.html
  • Ensure public access is enabled via policy document
