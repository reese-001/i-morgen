aws:dynamo db:commands

Create Table

Two attributes with HASH (PK) and RANGE (SORT) defined

aws dynamodb create-table \
  --table-name Starships \
  --attribute-definitions \
   AttributeName=ShipClass,AttributeType=S \
   AttributeName=Registry,AttributeType=S \
  --key-schema \
   AttributeName=ShipClass,KeyType=HASH \
   AttributeName=Registry,KeyType=RANGE \
  --provisioned-throughput \
   ReadCapacityUnits=5,WriteCapacityUnits=5

Describe Table

output = table. Other option is 'text'. Default is json

aws dynamodb describe-table --table-name Starships --output table

Batch write to table

This assumes that the input file is in correct json

aws dynamodb batch-write-item --request-items file://batches/batch-000.json

Get item from table

Key file is json containing PK and Span values

aws dynamodb get-item --table-name Starships --key file://key.json

Delete item from table

Key file is json containing PK and Span values

aws dynamodb delete-item --table-name Starships --key file://key.json

Write item to table

Key file is json containing PK and Span values and all attribute data

aws dynamodb put-item --table-name Starships --item file://key.json --return-consumed-capacity TOTAL
