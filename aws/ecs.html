aws:ecs

  

Five components of ECS

  • Cluster - multiple ec2's that will house the docker container
  • Task Definition - json config that defines up to 10 containers
  • Task - launches containers defined in task definition - do not remain running after workload complete
  • Service - similar to task but remain running
  • Container agent - binary on each ec2 that monitors, starts and stops tasks
