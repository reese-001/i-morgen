Creating a Code Build Project

  • Assume a starting point of selecting "Create Build Project"
    • Not all options will be called out
  • Source
    • Options include (Feb 2023)
      • S3
      • CodeCommit
      • GitHub/Enterprise
      • BitBucket
    • Reference type
      • Branch
        • How to test ongoing code
      • Git Tag
        • For example, did you tag a commit to read "V1"? If so, you can select it here
      • Commit ID
        • Do you want to test a specific version of the repo
      • Additional configuration
        • This is more advanced and not yet reviewed
  • Environment
    • Environmental Image
      • Managed vs Custom
        • Do you need some specific software installed on your image to perform build and test
    • Additional Configuration
      • Note that the range of timeouts is from 5 min -> 8 hrs
        • This contrasts with Lambda with a timeout of 15 min which limits the ability to perform build and test ops
