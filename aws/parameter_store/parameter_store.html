Parameter Store

What does it do? 

  • Secure, hierarchical storage for configuation data management and secrets management
    • passwords
    • dataabase strings
    • license codes
  • Store configuation data and secure strings in hierarchies and track versions
  • Can be encrypted via KMS

Creation Details 

  • Name: can be grouped into hierarchies. This enables you to fetch all parameters at specific levels
  • Tier: see table

    • Standard Advanced
    # of params/region 10k 100k
    param policies 4kb 8kb
    cost free $.05 per param per month

  • Type

Parameter Policies 

  • Parameter Expiration
    • useful to force you to update or delete passwords
    • "I want this to expire at this date and time
    • ExpirationNotification can be set to notify prior to expiration
    • NoChangeNotification informs if a paramater has not been modified in a specific time period
  • Multiple policies can be applied per parameter
