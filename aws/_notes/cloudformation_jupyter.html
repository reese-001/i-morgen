Create CloudFormation script to generate a Jupyter Notebook

Source: ChatGPT

AWSTemplateFormatVersion: '2010-09-09'

Description: 'CloudFormation template to create a Jupyter notebook in AWS'

Resources:
  JupyterNotebook:
    Type: 'AWS::EMR::NotebookInstance'
    Properties:
      Name: 'MyJupyterNotebook'
      RoleArn: 'arn:aws:iam:::role/'
      DirectInternetAccess: 'Enabled'
      VolumeSizeInGB: 10
      InstanceType: 'ml.t2.medium'
      SubnetId: ''
      SecurityGroupIds:
      - ''
      DefaultCodeRepository: 'https://github.com//.git'
  • You will need to replace the following:
    • YOUR_AWS_ACCOUNT_ID: Your AWS account ID.
    • EMR_NOTEBOOK_ROLE: The ARN of the IAM role that you created for the EMR Notebook.
    • YOUR_SUBNET_ID: The ID of the subnet in which to launch the EMR Notebook.
    • YOUR_SECURITY_GROUP_ID: The ID of the security group to associate with the EMR Notebook.
    • YOUR_GITHUB_USERNAME: Your GitHub username.
    • YOUR_REPO: The name of your repository on GitHub.
