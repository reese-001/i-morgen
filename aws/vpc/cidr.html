CIDR (Classless Inter-Domain Routing)

Contents 

  • What is it
  • How do they work?
  • Private vs Public IP's

    • What is it?

    • A method for allocating IP addresses
    • Used in Security Group rules and AWS (and GCP, Azure and others) networking in general
    • Help define IP ranges
      • ww.xx.yy.zz/32 => one IP
      • 0.0.0.0/0 => all IP's

    How do they work?

    • Consists of two components
      • Base IP
        • Represents an IP contained in the range (xx.xx.xx.xx)
        • Examples: 10.0.0.0, 192.168.0.0, ...
      • Subnet Mask
        • Defines how many bits can change in the IP
        • Examples: /0, /24, /32
      • Examples of subnet masks
        • 192.168.0.0/32 = 192.168.0.0 (loopback)
        • 192.168.0.0/31 = 192.168.0.0 - 192.168.0.1
        • 192.168.0.0/30 = 192.168.0.0 - 192.168.0.3
        • 192.168.0.0/29 = 192.168.0.0 - 192.168.0.7
        • 192.168.0.0/24 = 192.168.0.0 - 192.168.0.255
        • 192.168.0.0/16 = 192.168.0.0 - 192.168.255.255

    Private vs Public IP's

    • Private IP's
      • 10.0.0.0 - 10.255.255.255 (10.0.0.0/8)
      • 172.16.0.0 - 172.31.255.255 (172.16.0.0/12)
      • 192.168.0.0 - 192.168.255.255 (192.168.0.0/16)
    • All others are public
