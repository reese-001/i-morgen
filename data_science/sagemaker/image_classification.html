    • What Is It

    • Background
      • Similar to object detection, however it does not provide the bounding boxes or additional details
      • Assigns one or more labels to an image
    • Training Data
      • Apache MXNet RecordIO
        • Not Protobuf
        • This ensures interoperability with other deep learning frameworks
      • Raw jpb/png files
        • Requires "lst" files to associate the image index with a class label and path to image
      • Augmented Manifest Image Format
        • Can be used with Pipe mode
      • Example of "lst" file
        • 5 1 your_image_directory/train_img_dog1.jpb
        • 1000 0 your_image_directory/train_img_cat1.jpb
        • 22 1 your_image_directory/train_img_dog2.jpb
    • How is it used
      • Uses ResNet CNN
      • Two modes
        • Full training mode
          • Network initialized with random weights
        • Transfer learning mode
          • initialized with pre-trained weights
          • The top fully connected layer is initialized with random weights
          • The idea is to wipe out that top layer and recreate it with the new learnings
        • Default image size is 3 channel 224x224
    • Hyperparameters
      • Batch size, learning rate, optimizer
      • Optimizer-specific parameters
        • Weight Decay
        • Beta 1
        • Beta 2
        • EPS
        • Gamma
