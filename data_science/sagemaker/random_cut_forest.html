    • What Is It

    • Used in anomaly detection to detect unexpected spikes in time series data
      • Breaks in periodicity
      • Unclassifiable data points
      • Assigns anomaly score to each data point
      • This anomaly score is generated by the complexity of the trees that result from analyzing the data
    • Is an unsupervised model

    Usefulness

    • Can RCF be used to detect incident ticket anamolies?
      • As a machine learning algorithm for anomaly detection, RCF can be applied to any type of data where unusual patterns or deviations are of interest, including incident tickets
      • By analyzing patterns in the data, RCF can identify tickets that deviate from the normal behavior and flag them as potentially problematic
      • This can be useful for IT departments in identifying incidents that require immediate attention or for identifying patterns that may indicate a larger issue that needs to be addressed

    Training Data

    • RecordIO-protobuf
    • CSV
    • Can optionally create a test channel for computing accuracy, precision, recall, F1 on labelled data

    How Is It Used

    • Creates a forest of trees where each tree is a partition of the training data
    • The resulting change in complexity of the tree as a result of adding a point into it
    • Data is randomly sampled

    Hyperparameters

    • Num_trees: more trees reduces noise
    • Num_samples_per_tree: Should be chosen such that 1/num_samples_per_tree approximates the ration of anomalous to normal data

    Instance Types

    • Does not take advantage of GPU's
