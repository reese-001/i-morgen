    • Background
      • Identify all objects in an image with bounding boxes
      • Detects and calssifies objects with a single deep neural network
      • Classes are accompanied by confidence scores
      • Can train from scratch or use pretrained models based on ImageNet
    • What kind of training input does it expect
      • RecordIO or image format (jpg pr png)
      • With an image format, supply a JSON file for annotation data for each image
        • review image types
      • Must identify where each item of interestg is on the picture and tell Object Detection what it is
    • How is it used
      • Takes an image as input and outputs all instances of objects in the image with categories and confidence scores
      • It uses a CNN with Single Shot MultiBox Detector algorithm
        • The base CNN can be VGG-16 or ResNet-50
      • Allows transfer/incremental learning
      • Uses Flip, rescale, and jitter internally to avoid overfitting
    • Hyperparameters
