    • The question is "How do we implement these concepts using EC2 or Elastic Map Reduce (EMR)?
    • EMR supports Apache MXNet out of the box and different GPU instance types
    • Consider the following instance types to gain access to GPU's
      • P3: 8 Telsa V100 GPU's
      • P2: 16 k80 GPU's
      • G3: 4 M60 GPU's
      • P2/P3 are typical choices
    • AWS has made deep learning AMI's available
      • Come preinstalled with MXNet or TensorFlow (or others?)
      • Pair that with a P2 or P3 and you are ready to go
