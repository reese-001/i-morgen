  • What Is It

    • What Is It

    • Managed service provided by AWS that enables the training and deployment of highly accurate natural language processing (NLP) models, specifically for the task of supervised text classification
    • Two primary purposes
      • Text classification
        • Predict labels for sentences
        • Useful for web seaches and information retrieval
      • Word2vec
        • Creates a vector representation of words
        • Semantically similar words are represented by vectors close to each other
        • This is known as word embedding

    • What kind of training input does it expect?
      • Regardless of the format, the text requires tokenization and should be lowered
      • For supervised mode (text classification)
        • One sentence per line
        • First "word" in the sentence is the string "__label__" followed by the label
        • ex. __label__1 This is my sentence
        • Alternatively it accepts augmented manifest text format
          • It is a JSON file that specifies the location of the training data, as well as the labels and other metadata associated with each example
      • For Word2vec
        • Requires a text file with one training sentence per line
        • ex. {"source":"This is my sentence"}

    • Word2vec has multiple modes
      • Cbow
      • Skip-gram
      • Batch skip-gram
        • Distributed computation over many CPU nodes

    • Hyperparameters
      • Word2vec
        • Mode (batch_skipgram, skipgram, chow)
        • Learning_rate
        • Window_size
        • Vector_dim
        • Negative_samples
      • Text classification
        • Epochs
        • Learning_rate
        • Word_ngrams
        • Vector_dim
