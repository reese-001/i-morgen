  • What Is It

    • What Is It

    • Used for training object-based embeddings on large datasets
    • Built on top of the open-source library Gensim
    • It is available as part of the AWS SageMaker platform and uses distributed computing and GPU acceleration to speed up training and make it more efficient
    • Object2Vec can be used for tasks such as item-to-item similarity, item-to-set similarity, and clustering

    • Can be used for the following
      • Compute neatest neighbors of objects
      • Visualize clusters
      • Genre prediction
      • Recommendations

    • What kind of input does it expect?
      • Data must be tokenized into integers
      • Training data consists of pairs of tokens and/or sequences of tokens
        • Sentence-sentence
        • Labels-sequences
        • Customer-customer
        • Product-product
        • User-item

    • How is it used?
      • Process data into JSON lines and shuffle it
      • Train with two input channels, two encoders, and a comparator
        • Comparator function takes two input embeddings (vectors) and returns a similarity score between 0 and 1, indicating how similar the two objects are
      • Encoder choices
        • Average-pooled enbeddings
        • CNN's
        • Bidirectional LSTM
      • Comparator is followed by a feed-forward neural network
