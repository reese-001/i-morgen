    • Glue Data Catalog

    • Fully managed and centralized repository for storing the metadata of the data stored in data stores such as Amazon S3 and Amazon Redshift
    • The metadata includes information about the structure of the data, the relationships between tables, and the schema information of the data
    • Metadata repository for all your tables
    • Integrated with Athena or Redshift Spectrum to do schema and data discovery
    • Makes use of Glue Crawlers to help build the Glue Data Catalog

    Crawlers

    • Crawlers go through your data to infer schemas and partitions
    • Supports various data formats such as CSV, JSON, Parquet, and Avro, as well as data stored in databases like Amazon RDS and Amazon Redshift
    • It can also crawl semi-structured data sources like XML and HTML
    • Once the AWS Glue crawler has completed the crawl, the resulting table can be used in AWS Glue jobs for transforming and loading data
    • The schema information in the AWS Glue Data Catalog is used to determine the data types, relationships, and mapping between source and target data
    • Crawlers can be run on a schedule or on demand
    • Need an IAM role/credentials to access the data stores

    Partitions

    • Crawler will extract partitions based on how your S3 data is organizationed
    • For example
      • Do you query by time ranges? If so, consider organizing your buckets like 's3://my-bucket/dataset/yyyy/mm/dd/device'
      • Do you query by deviced? If so, consider organizing your buckets like 's3://my-bucket/dataset/device/yyyy/mm/dd'

    ETL

    • AWS Glue is a fully managed ETL (Extract, Transform, Load) service offered by Amazon Web Services
      • It makes it easy to move data between data stores
      • It can automate the discovery of data sources
      • Extract data from various sources
      • Transform data into the format required by analytics and business applications
      • And load the data into destinations like data lakes and data warehouses
    • ETL code can be created in Python or Scala
    • Can provide your own Spark or PySpark scripts
    • Fully managed and cost effective

    ETL Transformations

    • Bundles transformations
      • DropFields
      • DropNullFields
      • Filter
      • Join
      • Map
    • Machine learning tranformations
      • FindMatches ML
        • Identifies duplicate or matching records in your dataset, even if the records do not have a common identifier and no fields match exactly
    • Format conversions
      • CSV
      • JSON
      • Avro
      • Parquet
      • ORC
      • XML
