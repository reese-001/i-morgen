Ensemble Methods: Bagging and Boosting

Contents

  • Overview
  • Bagging
  • Boosting

    • Overview

    • Ensemble model takes multiple models (or multiples of a single type of model) and lets them vote on the result
    • For example, random forest
      • Consists of many decision trees with slightly different turning or datasets
    • How these trees differ is what defines the difference between bagging and boosting

    Bagging

    • Generate new training sets by random sampling with replacement
    • Each resampled model can be trained in parallel

    Boosting

    • Observations are weighted
    • Some will take part in new training sets more often
    • Runs in a serial fashion

    Bagging vs Boosting

    • XGBoost is hot and generally produces better accuracy
    • Bagging avoids overfitting
    • Bagging is faster (ie can run in parallel
