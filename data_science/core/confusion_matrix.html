Confusion Matrix


Contents
  • Overview

    • Overview

    • Explains the more nuanced performance metrics of your model
    • Brings into view true positives, true negatives, false positives and false negatives
    • Examples of why this is useful
      • A test for a rare disease can be 99.9% accurate simply by guessing "no" all the time
    • There is no real convention on the format of these matrixes. Ensure you look at the row and column headers to understand what it is saying
    • Most of your results will be in the diagonal (true positives, true negatives) - this is where accuracy lives
    • Different types of confusion matrixes
      • Some confusion matrixes have totals included. See link
      • Others will be multi class. See link
