    • Introduction

    • Pandas
      • Provides data structures for efficiently storing large datasets and tools for working with them, including reading and writing data from various sources such as CSV and Excel files, and performing statistical analysis on the data
    • Numpy
      • Provides support for large, multi-dimensional arrays and matrices of numerical data, and has a large collection of functions for performing mathematical operations on these data structures
    • Matplotlib
      • Allows for the creation of line plots, scatter plots, bar plots, error bars, bar charts, histograms, scatterplots, etc
    • Seaborn
      • Data visualization library in Python that is built on top of matplotlib and provides a higher-level interface for creating and customizing plots
      • Unique outputs:
        • Heat map
        • Pair plot
        • Joint plot
    • scikit-learn
      • Library for machine learning in Python that provides a wide range of tools for building and training machine learning models, as well as tools for evaluating the performance of those models and for making predictions using the trained models

    EDA Framework

    • Review data for unexpected-ness in the following (non exclusive) areas
      • Column names
      • Missing data
      • Outliers
      • Execute dataframe.describe()
    • Respond to missing data
      • Drop
        • Take steps to avoid introducing bias when dropping data
        • dataframe.dropna()
      • Impute
    • Convert data that exists in Pandas dataframe into numpy array
    • Normalize data
      • Ensures data is centered around the mean for each column and scaled down to the same range so they have the same weight
        • from sklearn import preprocessing
        • scaler = preprocessing.StandardScalar()
        • features_scaled = scaler.fit_transform(features)

    Data Types

    • Numerical
      • Represents some sort of quantitative measurement
        • Height
        • Page load times
        • Stock prices
      • Two kinds of numerical data
        • Discrete Data
          • Integer based, often counts of some event
            • How many purchases did a customer make in a year?
            • How many times did I flip heads
        • Continuous data
          • Has an infinite number of possible values
            • How much time did it take for a user to check out?
            • How much rain fell on a given day
    • Categorical
      • No inherent mathematical meaning
        • Gender
        • Yes/no
        • Race
        • State of residence
      • You can assign numbers to the categories, but they have no inherent meaning
        • Example: the size of Texas is greater then the size of Florida, but Texas is not greater then Florida
    • Ordinal
      • Mixture of numerical and categori
      • Categorical data that has mathematical meaning
        • Movie ratings, 1 means its worse then 2

    Data Distributions

    • [For Reference] There are two types of functions
      • Probability Density Function
        • A probability density function (PDF) is a way to describe the likelihood of a continuous variable having a certain value
        • It is represented by a curve, where the height of the curve at any point represents the probability of the variable taking on that value
        • The area under the curve over a range of values represents the probability that the variable falls within that range
        • PDF example: link
        • Examples: Normal, Triangular, Lognormal, Beta
      • Probability Mass Function
        • A probability mass function (PMF) is similar to a probability density function (PDF), but it is used to describe the probability of a discrete random variable
        • Looks a lot like a histogram - because it is a histogram
        • PMF example: link
    • Normal distribution
      • This is an example of a PDF
      • For a given range of values, it gives you a way to find the probability of that range occuring
      • Divided into deviations
    • Poisson distribution
      • This is an example of a PDF
      • A discrete probability distribution that is used to model the number of events that occur in a fixed interval of time or space, given that the average number of events per interval is known.
      • Poisson example: link
    • Binomial distribution
      • Another example of a PDF
      • Describes the number of successes in a sequence of experiments with a yes or no question
      • Example is flipping a coin
      • A specific case of the Binomial distribution is the Bernoulli Distribution
        • Has a single trial (n = 1)
        • Can think of Binomial distribution as the sum of Bernoulli distributions

    Time Series: Trends and Seasonality

    • Discrete points in time over a period of time
    • Trends
      • The overall direction that the data points are moving in over time
      • Can be upward (positive), downward (negative) or flat (horizontal or flat)
    • Seasonality
      • Seasonality refers to the presence of regular, repeating patterns within the data that occur over specific, predictable time periods, such as daily, weekly, monthly, or yearly.
      • Trend and Seasonality can co-exist in a time series
      • Seasonality can be removed for a clearer understanding of trends
    • Noise
      • Some variations are just random in nature
      • Two ways to model with noise
        • Additive model
          • Seasonality + Trends + Noise = time series
          • Characteristic: seasonality is constant
        • Multiplicative model
          • Characteristic: seasonality scales with trend
