Linear Regression

What is it?

  • Linear regression is a statistical method used to model the relationship between a dependent variable and one or more independent variables
  • The dependent variable is predicted from a linear combination of the independent variables
  • The goal of linear regression is to find the best fitting straight line through the data points, so that we can use this line to make predictions about the dependent variable based on the values of the independent variables
  • This line is called the regression line
  • Linear regression is used to model continuous variables, such as weight or temperature, rather than discrete variables like yes or no, or true or false.
