    • PNG

    • Summary
      • Lossless format: Preserves all the original image data and quality when it is saved
      • It uses a type of compression called DEFLATE, which is efficient for images with large areas of the same color
    • Good for
      • PNG images are best used for images with sharp lines and large areas of the same color, such as logos and icons
      • Transparent backgrounds
        • PNG files support transparency, meaning you can have parts of the image that are transparent, which is useful for overlaying images or for creating transparent backgrounds
      • Logos and icons
        • PNG is great for images with sharp lines and large areas of the same color, such as logos and icons. It will preserve the sharp edges of the image and maintain the integrity of the image's colors
      • Screenshots and screen grabs
        • PNG is useful for screenshots and screen grabs because it preserves the quality of the original image and allows you to retain the same level of detail.

    JPEG

    • Summary
      • Lossy format: Discards some of the original data when it is saved to achieve smaller file size
      • It uses a different type of compression that is better suited for images with many colors and variations in color, such as photograph
    • Good for
      • JPEG images are best used for photographs and other images that require a smaller file size.
      • Prefereed for web sites due to smaller size and faster load times
      • Photographs and other images with many colors and variations in color
        • JPEG uses a type of compression that is better suited for images with many colors and variations in color, such as photographs. It can produce smaller file sizes while still maintaining a good level of image quality
      • Images for email attachments
        • Due to its smaller file size, JPEG is a good option for sending images as email attachments.
