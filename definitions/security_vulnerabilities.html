    • Cross Site Scripting (XSS)

    • Type of security vulnerability that allows an attacker to inject malicious code into a web page viewed by other users
    • The injected code can steal sensitive information such as passwords, or perform actions on behalf of the victim, such as making unauthorized purchases
    • XSS can occur when a web application does not properly validate user input and includes it in the generated web page without proper encoding or filtering
    • Example
      • In comments section, place a script block with a src tag that points to a Browser Exploitation Framework Project (BeEF) url
      • Log into BeEF and view open connections to that web site
      • Push login requests to a specific user
        • For example, Facebook or Google
        • User will be prompted to login to that web site and BeEF will share the password with you
