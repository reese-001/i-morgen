ACID atomicty consistancy isolation durability
Blackbox testing a method of testing software where the tester does not have access to the internal structure of the code being tested. The tester interacts with the software purely through its external interfaces, such as a graphical user interface or an API, and has no knowledge of the underlying code.
Certificate Pinning a security measure that involves associating a host with its X509 certificate or public key and storing it locally on the client device. The client can then verify the identity of the host by comparing the stored certificate or public key to the one presented by the host during SSL/TLS negotiation, helping to prevent man-in-the-middle attacks.
Control Plane Responsible for the management and coordination of the system. This includes tasks such as making decisions about how to distribute and process data, monitoring the state of the system, and providing an interface for external management and configuration.
Data Plane Responsible for the actual processing and handling of data. This includes tasks such as receiving, transforming, and forwarding data, as well as enforcing data validation and access control rules.
Endianness Used to describe the order in which the bytes of a multi-byte value are stored in memory or transmitted over a network. Little endian stores the LSB in the lowest memory address while big endian stores the MSB in the lowest address
Marshalling Data The process of transforming the memory representation of an object into a data format suitable for storage or transmission. It is typically used when data must be moved between different parts of a computer program or from one program to another.
Piece Wise Constant Function that is defined by multiple constant values over different intervals of the input domain. The output of the function remains constant within each interval, and changes abruptly at the boundaries between intervals.
POSIX file system File system that is widely used in many different types of computer systems and are known for their stability, reliability, and portability. They are often used in environments where multiple users need to access and share files and resources, and are well suited for use in server systems, networked environments, and other types of distributed systems.
Race condition Occurs when the program's behavior is determined by which event occurs first
Rectification The process of thresholding the negative values of a signal to 0
White box testing A method of testing software where the tester has access to the internal structure of the code being tested. This allows for a thorough examination of the code, including testing of individual functions, statements, and branches, as well as the interactions between them.
Snowflake Servers Refers to a server instance that is unique and individualized, often customized and configured manually, making it difficult to replicate or replace. Opposite of modern devops/IaC practices
