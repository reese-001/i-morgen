    • Overview

    • Git branches are effectively a pointer to the snapshot of your related changes
      • In other words, instead of copying files from directoy to directory, git stores each branch as a reference to a commit
      • A branch is not a container for commits but rather represents the latest in a series in related commits

    Creating Branches

    • Remember that branches are simply pointers to commits
    • When you create a new branch, Git simply creates a new pointer. No other changes are made to the repository
      • Specifically, "git branch new-branch" creates the branch but there are no commits associated to that branch yet
    • To start adding commits, you need to select it with a "checkout command" and then standard "add" and "commit" commands
      • Specifically, "git checkout new-branch", then "git add" and finally "git commit"
    • The above commands create local branches, however the same ideas apply to creating remote branches
      • Add a new remote repository to your local repository's configuration
          • git remote add new-remote-repo https://bitbucket.com/user/repo.git
      • Push the branch to the new-remote-repo
        • git push new-remote-repo new-branch

    Deleting Branches

    • Once the branch has been merged into the main code base, it can be safely deleted from the local repository
      • git branch -d "branch-name"
    • However, if the branch has not been merged, it will not be deleted and will generate a failure
      • error: The branch 'crazy-experiment' is not fully merged. If you are sure you want to delete it, run 'git branch -D crazy-experiment'
      • Can be bypassed with git branch -D "branch-name"
    • Deleting a remote branch is similar to deleting a local branch
      • git push origin --delete "branch-name"
      • git push origin :"branch-name"

    Commands

    Intent Command
    List all branches in your repository git branch --list
    Create a new branch called "branch" git branch <branch>
    Safe delete a branch called "branch" git branch -d <branch>
    Force delete a branch called "branch" git branch -D <branch>
    Rename the current branch to "branch" git branch -m <branch>
    List all remote branches git branch -a
