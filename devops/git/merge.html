Git Merge

  • Commands
  • Quizlet (external)

    • Overview

    • Intent
      • Putting a forked history back together again
      • Enables you to take the independent lines of development created by git branch and integrate them into a single branch
      • This often takes the form of integrating a feature branch into the main branch
    • Types of behavoirs
      • Fast forward merge
        • If the branch you are merging into has no new changes since the branch you are merging from was created, a fast foward merge will be done
        • Simply movesthe current branch pointer to the same commit as the branch being merged
        • No new commit created
          • Alternatively, force a new commit by using the "--no-ff" option
        • Picture: before
        • Picture: after
      • Three way merge
        • If both branches have new changes, a three way merge will be performed
        • Git will attempt to combine the changes from both branches and create a new commit that represents the merged changes
        • Picture: before
        • Picture: after
      • Merge conflicts
        • If Git is unable to automatically merge, you will need to resolve the conflicts manually
        • Git will mark the conflicted files in your working directory and you will need to edit
        • When complete, you can commit the changes to complete the merge
      • Merge commit
        • When a merge is complete, Git will create a new commit that represents the merged changes
        • This commit will have two or more parent commits, representing the branches that were merged
        • Picture: before
        • Picture: after

    Walk Through

    • Fast Forward Merge: Create new branch, add commits, integrate it back into main line
      • ## start new feature
      • git checkout -b new-feature main
      • ## edit file
      • git add &ltfile>
      • git commit -m "Start a feature"
      • ## edit file
      • git add &ltfile>
      • git commit -m "Finish a feature"
      • ## merge into a new feature branch
      • git checkout main
      • git merge new-feature
      • git branch -d new-feature

    Commands

    Intent Command
    Standard merge git merge &ltbranch>
    Fast forward merge with commit git merge --no-ff &ltbranch>
