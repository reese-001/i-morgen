Network Protocols
  • A network protocol is a set of rules and standards that govern the communication between devices on a network
  • Network protocols are necessary for devices on a network to understand how to communicate with each other
  • Examples include
    • IP
    • TCP
    • HTTP
    • FTP
    • DNS
    • DHCP
    IP
    • Internet Protocol
    • Function is to route packets of data between devices on a network
    • These data packets are known as "IP Packets" and are the fundumental unit of data that gets sent from one device to another
    • IP Packets consist of two segments
      • IP Header
        • Size is typically between 20-60 bytes
        • Contains
          • Source IP
          • Destination IP
          • Internet Header Length (IHL)
          • Version (IPv4 or IPv6)
      • Data
        • The total size of the IP packet is 2^16 bytes
        • To meaningfully transmit data, you will need a lot more then 2^16 bytes of data capacity
        • Additionally, because IP is connection-less and best effort, it cannot guarantee delivery of the packet nor the delivery in the correct order
        • This introduces the functionality of TCP (and UDP)
    TCP
    • Transmition Control Protocol
    • Built on top of IP and introduces the concept of 'connections'
    • Intended to solve the following problems
      • Deliver packets in ordered manner
      • Deliver packets reliabily - or at least inform if they are not able to be delivered
      • Deliver packets without corruption
    • These allow the transmission of arbitrarily long data
    • Introduces the TCP header to the IP Packet. This lives alongside the IP Header
    • Connections originate from a three way handshake - aka SYN, SYN-ACK, ACK process
      • First device says "I want to connect with you" (SYN)
      • Second device responds "I accept your request" (SYN-ACK)
      • First device responds with "We are now connected" (ACK)
    • Related discussion on Performance Engineering link
    • Timeouts are defined by a period of inactivity
